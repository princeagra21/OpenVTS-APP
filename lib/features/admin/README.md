# Admin Feature

Admin role screens and features for day-to-day fleet operations.

## Structure
```text
lib/features/admin/
  components/          # admin widgets and feature components
  screens/             # admin screens and flows
  router/              # admin route list
  layout/              # admin shell/layout
```

## Responsibilities
- Fleet management: vehicles, drivers, users, teams
- Operational settings
- Activity tracking
- Payments, renewals, plans, SIMs, and devices
- Support workflows
