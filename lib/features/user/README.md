# User Feature

End-user experience feature for fleet customers and lower-level roles.

## Structure
```text
lib/features/user/
  components/          # user feature components
  screens/             # user screens and flows
  router/              # user route list
  layout/              # user shell/layout
```

## Responsibilities
- Profile and settings
- Vehicle access and usage
- Live tracking and map views
- Notifications and support
