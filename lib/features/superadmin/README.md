# Superadmin Feature

The Superadmin feature powers platform-level management: onboarding, dashboards, system settings, payments, support, vehicle management, and server status.

## Structure
```text
lib/features/superadmin/
  components/          # superadmin feature components
  screens/             # superadmin screens and flows
  router/              # superadmin route list
  layout/              # superadmin shell/layout
```

## Key Screens
- Dashboard
- Admins
- Vehicles
- Payments
- Support
- Server status
- Platform settings
